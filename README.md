# Registro de Estudiantes

Proyecto web con HTML, CSS y JavaScript para registrar estudiantes, validar el formulario y actualizar una tabla sin recargar la pagina.


## Funcionalidades

- Lista inicial con Maria Mora Perez, Pedro Sibaja Lopez y Marco Rojas Castro.
- Validacion de campos obligatorios.
- Validacion de nota entre 0 y 100.
- Agregado dinamico de estudiantes.
- Limpieza automatica del formulario.
- Nota en verde si es mayor o igual a 80.
- Nota en rojo si es menor a 65.
- Resumen actualizado con mayor nota, promedio y nota mas baja.
- Si varios estudiantes tienen la mayor nota, se muestran todos en el resumen.
